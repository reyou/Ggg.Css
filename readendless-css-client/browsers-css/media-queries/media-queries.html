Using media queries

@media (hover: hover) { .mouseOnlyStyle { background-color: red; } }
@media (hover: none) { .mouseOnlyStyle { background-color: green; } }

This div will have red background only devices with mouse input, if this page will be rendered by a mobile (touch screen) device then background will be green.

Following will be visible under 640px.
This is visible, because it is size is under 640px.

@media (min-width: 30px) and (orientation: portrait) { ... }
